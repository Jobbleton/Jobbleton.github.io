Drone Mapping

Company-Sponsored Project

This project relies on a quadcopter to map a local area decided by a user on an Android device. The resulting data is rendered back to the device as a KML file.

Built upon existing architecture, our team designed and implemented the mapping component and updated the autopilot code.
